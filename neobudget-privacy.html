Politique de confidentialité — NeoBudget

Dernière mise à jour : 30 octobre 2025

Cette politique de confidentialité décrit comment l’application NeoBudget, développée par DMC Pixel, traite vos données et respecte votre vie privée.

1. Données collectées

NeoBudget ne collecte, ne partage et ne transmet aucune donnée personnelle. Toutes les informations que vous saisissez (revenus, dépenses, budgets, épargne) sont enregistrées uniquement sur votre appareil, dans une base de données locale (Hive).

2. Publicités

L’application utilise Google AdMob pour afficher des publicités occasionnelles. Google peut collecter certaines données techniques anonymes (identifiant publicitaire, type d’appareil, version Android) pour proposer des annonces adaptées. Ces données ne permettent pas de vous identifier personnellement.

Vous pouvez consulter la politique de confidentialité de Google ici : https://policies.google.com/privacy?hl=fr

3. Droits de l’utilisateur

  • Vous pouvez effacer toutes vos données via le bouton “Réinitialiser les données” dans les paramètres.
  • Vous pouvez désinstaller l’application à tout moment pour supprimer définitivement toutes les informations locales.

4. Sécurité

Toutes les données restent sur votre appareil et ne sont pas accessibles par d’autres applications. NeoBudget ne nécessite pas de connexion Internet pour fonctionner (sauf pour afficher les publicités).

5. Contact

Pour toute question relative à la confidentialité ou à vos données, contactez-nous à : dmcpixel3@gmail.com
